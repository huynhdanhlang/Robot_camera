## Prepare
- Arduino (Mega,Uno,...)
- Module L298N driver DC motor
- 2x DC motor(geared motor mini)
- Module ZC-A0591 driver Step Motor
- 1x Step motor
- 1x Switch
- 1x power blank
- Jumper Wire
- Mini breadboard
- Car chassis and wheels
- Bluetooth HC-05
- 2x Smartphone
